# SC2 Hotkeys

A custom StarCraft II hotkey profile designed around QWER for abilities and A, S, Z, X, C for core movement commands.

The goal is to keep similar actions on the same keys across all three races, campaign, and Co-op commanders whenever possible.

```text
                      SC2 Hotkey Layout

┌────────────┬────────────┬────────────┬────────────┐
│     Q      │     W      │     E      │     R      │
├────────────┼────────────┼────────────┼────────────┤
│ Ability 1  │ Ability 2  │ Ability 3  │ Ability 4  │
└────────────┴────────────┴────────────┴────────────┘

    ┌────────────┬────────────┬────────────┬────────────┐
    │     A      │     S      │     D      │     F      │
    ├────────────┼────────────┼────────────┼────────────┤
    │   Attack   │    Stop    │ Ability 5  │ Ability 6  │
    └────────────┴────────────┴────────────┴────────────┘

       ┌────────────┬────────────┬────────────┐
       │     Z      │     X      │     C      │
       ├────────────┼────────────┼────────────┤
       │    Move    │ Hold Pos.  │   Patrol   │
       └────────────┴────────────┴────────────┘
```


## Design Philosophy
### 1. Consistent Core Commands

Core unit commands never change.

| Action | Key |
|--------|-----|
| Move | **Z** |
| Stop | **S** |
| Hold Position | **X** |
| Patrol | **C** |
| Attack | **A** |

These five commands are intended to become permanent muscle memory.

---

### 2. QWERTY for Abilities

Most active abilities are placed on the **QWERTY** row.

General pattern:

| Purpose | Key |
|---------|-----|
| Primary ability | **Q** |
| Secondary ability | **W** |
| Utility | **E** |
| Tactical/Special | **R** |
| Toggle/Halt | **T** |

Examples include:

- Stimpack → **Q**
- Blink → **Q**
- Force Field → **Q**
- Psi Storm → **W**
- Repair → **E**
- Spray → **R**

---

### 3. D/F for Secondary Interactions

Frequently used secondary commands are grouped together.

| Action | Key |
|--------|-----|
| Gather | **D** |
| Return Cargo | **F** |
| Load Nearby Units | **D** |
| Tech Lab | **D** |
| Reactor | **F** |
| Cloak | **D** |
| DeCloak | **F** |

---

### 4. Building Layout

Building placement is inspired by Grid but modified for better consistency.

Common patterns:

- Basic build menu → **Q**
- Advanced build menu → **W**
- Buildings generally remain close to their Grid positions

## Features

- Built for minimal finger movement and consistent muscle memory
- Supports Multiplayer, Campaign, and Co-op
- Covers nearly every unit and ability in StarCraft II
- Consistent hotkeys across all three races
- Easy transition between races and game modes

## Installation

1. Download the `.SC2Hotkeys` file.
2. Copy it to your StarCraft II Hotkeys folder.


Typical locations:

### Windows

```
Documents\StarCraft II\Accounts\<Account ID>\<Profile ID>\Hotkeys\
```

E.G. 
```
C:\Users\Desktop\Documents\StarCraft II\Accounts\123\Hotkeys
```

Hotkey path for offline games (e.g. when accessed through map editor)
```
C:\Users\USERNAME\Documents\StarCraft II\Hotkeys
```

(depending on your installation)

3. Launch StarCraft II.
4. Open **Options → Hotkeys**.
5. Select this profile.
